# Snackada Website

A professional business website for Snackada - Quality Meals. Barkada Deals.

## About Snackada

**Tagline:** Quality Meals. Barkada Deals  
**Description:** Ang snacks na pang-solo, partner, pamilya at barkda.

## Installation

1. Clone or download this repository
2. Place files in your web server directory (e.g., xampp/htdocs)
3. Ensure logo files are in `assets/images/logo/` directory:
   - primary.png
   - secondary.png
4. Open index.html in your browser
